# 2024/1/18

## 갑자기 서핏?

지인이랑 포트폴리오에 관련된 얘기를 하는 도중에

노션으로 포폴을 만드는 데 썸네일을 만들고 있었다.

라는 얘기를 하다가 서핏에 대해 알려주었다.

서핏이라고 포트폴리오에 쓰기 좋은 템플릿이 있었다고 참고해보라는 얘기가 나왔다.

## 서핏이란?

[서핏](https://www.surfit.io/)

<img width="1280" alt="image" src="https://github.com/GSMIOTjgh/portfolio/assets/132252115/c3c76d9a-bf01-43fe-922d-db01d0d1a16f">

웹개발, React, UI/UX, 스타트업 채용 등

개발자와 디자이너 부류에게 도움되는 정보 사이트 라고 생각한다(나는)

## 내 포트폴리오는?

내 포트폴리오는 기본적으로 노션을 이용한 노션 포트폴리오이다.

노션에는 링크, 임베드, 미리보기, 이모지 등

포트폴리오에 좋은 기능들이 있고 그 밖에 기능들이 많아 사용하기 편리하다는 점이 있다.

## 서핏 x 노션 포트폴리오

서핏 검색창에 [**노션 포트폴리오**](https://www.surfit.io/search/%EB%85%B8%EC%85%98%20%ED%8F%AC%ED%8A%B8%ED%8F%B4%EB%A6%AC%EC%98%A4)를 쳤을 때

위 링크에 들어가면 노션 포트폴리오에 대한 정보 팁 등을 서핏에서 찾아 볼 수 있다.

[**개인적인 추천**](https://brunch.co.kr/@famelee/18)
