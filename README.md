## 바뀐 것

1. **날 알아볼 수 있는 곳**에 velog 추가

2. 능력 => **사용 가능 언어 및 도구**로 재명칭

3. **사용 가능 언어 및 도구**에 프론트와 디자인 관련만 넣음(원래는 java같은 그런 것도 있었음)

4. **프로젝트**에 있던 프로젝트 종류가 바뀜
